طراحی (یا بازطراحی) یک اپلیکیشن تحت‌وب حرفه‌ای برای مدیریت کارهای روزانه با تمرکز بر یادگیری علمی و مرور منظم بر پایه‌ی تقویم هجری شمسی.


---

✅ وضعیت فعلی:
من فایل HTML اولیه را دارم که شامل طراحی ظاهری با TailwindCSS، تب‌های افقی بالا، فرم افزودن تسک، قابلیت مرور و تحلیل آمار است. ظاهر سایت ساده، دقیق، مدرن، و کاملاً مناسب استفاده شخصی است — و باید کاملاً حفظ شود.


---

🛠 تغییرات کلیدی مورد نیاز:
اضافه کردن تب ها البته تب های خالی با اسم های مستعار مثلا تب اضافه ۱ تب اضافه دو
چون من قراره در آینده صفحات جدیدی به سایت اضافه کنم برای همین نیاز دارم مثلا جوری باشه که وقتی روی اون تب خاص میزنم وارد یه صفحه جدید بشه که بعدا می‌سازم و آدرس میدم

هر آیتم منو باید به صورت کادر قابل کلیک با افکت فعال (active) باشد.


2. تاریخ شمسی

تمام تاریخ‌ها باید به‌صورت هجری شمسی ذخیره، نمایش و پردازش شوند.

هیچ‌گونه استفاده از تاریخ میلادی (new Date()، toLocaleDateString() و ...) مجاز نیست.

تاریخ باید شامل سال، ماه، روز (مثلاً ۱۴۰۴/۰۵/۲۲) باشد.

الگوریتم شمسی باید به‌صورت کاملاً بومی و بدون وابستگی خارجی پیاده‌سازی شود.


3. فرم افزودن تسک

فقط شامل عنوان و تاریخ هجری شمسی (روز / ماه / سال) و نوع فعالیت (مطالعه یا سایر).

هیچ فیلدی برای ساعت، روز هفته یا اولویت وجود نداشته باشد.

متن توضیحی فرم:

> «در چه تاریخی می‌خوای این کار رو انجام بدی؟ ما این روز رو به‌عنوان شروع یادگیری در نظر می‌گیریم و مرورها رو برات از همین روز برنامه‌ریزی می‌کنیم.»




4. مرور علمی خودکار

به‌محض ثبت تسک از نوع "مطالعه"، مرورهای زیر به‌صورت خودکار اضافه شوند:

روز مطالعه: روزی که ثبت شده برای اون کار 
مرور اول: ۱ روز بعد
مرور دوم: ۳ روز بعد 
مرور سوم: ۷ روز بعد 
با توجه به همون تحقیق علمی معتبر 
آزمون: قابل تنظیم دستی



ذخیره‌شده در قالب:

{
  title: "مطالعه فصل ۲ فیزیک",
  date: "۱۴۰۴/۰۵/۲۲",
  type: "مطالعه",
  reviews: [
    { type: "مرور اول", date: "۱۴۰۴/۰۵/۲۳" },
    { type: "مرور دوم", date: "۱۴۰۴/۰۵/۲۶" },
    { type: "مرور سوم", date: "۱۴۰۴/۰۵/۳۱" },
    { type: "آزمون",     date: "۱۴۰۴/۰۶/۰۷" }
  ]
}

5. تب‌ها و نمایش محتوا

برنامه امروز: فقط تسک‌هایی که تاریخشان برابر با تاریخ امروز است.

برنامه مرور: فقط مرورهایی که در تاریخ امروز هستند.

عملیات مجاز: تیک زدن، ویرایش، حذف، مشاهده جزئیات تسک مادر.


6. تنظیمات

قابل تنظیم توسط کاربر:

فاصله مرور اول، دوم، سوم، آزمون (تعداد روز)

حالت شب یا روشن


پشتیبان‌گیری: دانلود فایل JSON شامل همه تسک‌ها و مرورها.

بازیابی: بارگذاری فایل JSON ذخیره‌شده.

حذف کامل داده‌ها از حافظه مرورگر (LocalStorage).
